How It Works?
Your requested file's info are shared with url2drive's server
When you request for a file in the app, the app collects URL, the headers and cookies required for downloading the file. These information is sent to url2drive's server.
url2drive's server processes your request
The server retrieves the requested file and process it, so it can be brought to you using less Internet Data. This processed file is putted in your Google Drive Account. The Server checks that if you have sufficient free storage in your Google Drive account or not and notifies you if you dont have sufficient storage.
The App Downloads your file
The processed file is quickly downloaded by the app using your Google Drive Account.
The App brings your file in it's original form
After downloading, the app process the file to bring it back in it's original form and deletes the processed file form your Google Drive account.
