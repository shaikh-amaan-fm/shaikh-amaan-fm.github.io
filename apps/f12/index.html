Privacy Policy

F12 is an Android app that will provide you with options to inject custom JavaScript on webpages. You can interact in programming way with the webpages using the provided console. See the console logs, warning and Errors that webpages reports. Check the network traffics that a particular webpage is making. With Advanced Network mode enabled, you can get detailed information about the requests that webpages makes. Also, see the source code of currently loaded webpages on the go. Fetch all the medias that current webpage have loaded anytime. Privacy of the users are very important to us. Please review this privacy policy in order to understand how your data may get collected. This policy may change in future without any prior notice. However you may get notified about such changes by following @asfmapps on twitter.
